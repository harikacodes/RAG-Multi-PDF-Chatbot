# RAG Multi-PDF Chatbot

## Overview
AI-powered chatbot that allows users to upload multiple PDF documents and ask questions using Retrieval-Augmented Generation (RAG).

## Features
- Multi-PDF Upload
- PDF Text Extraction
- Text Chunking
- HuggingFace Embeddings
- FAISS Vector Database
- Semantic Search
- Gemini-Powered Question Answering
- Streamlit UI

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Google Gemini API

## Project Workflow
PDF Upload
→ Text Extraction
→ Chunking
→ Embedding Generation
→ FAISS Vector Store
→ Semantic Search
→ Gemini LLM
→ Answer Generation
